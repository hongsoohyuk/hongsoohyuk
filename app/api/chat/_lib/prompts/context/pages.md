## 페이지 사용법

### 홈 (/)

- 스크롤 애니메이션 히어로 타이틀과 네비게이션 카드가 있다.
- 각 카드를 클릭하면 해당 페이지로 이동할 수 있다.

### CLI 터미널 (/cli)

- 터미널 인터페이스로 포트폴리오를 탐색할 수 있는 재미 요소.
- 지원 명령어:
  - `ls` — 현재 디렉토리 파일 목록
  - `cd [dir]` — 디렉토리 이동
  - `cat [file]` — 파일 내용 출력
  - `pwd` — 현재 경로
  - `clear` — 화면 초기화
  - `help` — 사용 가능한 명령어 목록
  - `ask [질문]` — AI 채팅 열기
- 실제 Unix 쉘처럼 동작하며, 렉서/파서 기반으로 구현되어 있다.
- 터미널을 좋아하는 개발자에게 /cli를 추천한다.

### AI 채팅 (/chat)

- 지금 대화하고 있는 이 페이지.
- Gemini 모델 선택 가능 (Flash, Flash Lite, Pro).
- 수혁에 대한 질문, 프로젝트, 음악, 일상 등 무엇이든 물어볼 수 있다.

### 다국어 지원

- 모든 페이지는 한국어/영어를 지원한다.
- URL 앞에 /ko 또는 /en이 붙는다.
